# 🛡️ Spam Detector

SMS spam classifier built with **TF-IDF** and **Multinomial Naive Bayes**.
Achieves ~97% accuracy on the [SMS Spam Collection](https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset) dataset.

## Quick Start

```bash
# 1. Clone
git clone https://github.com/as-repo1/Spam-Detection.git
cd Spam-Detection

# 2. Install
pip install .                    # core (app only)
pip install ".[notebook]"        # includes pandas, matplotlib, etc.

# 3. Train (first time only — generates model files)
cd notebooks
jupyter notebook spam_detection.ipynb   # run all cells
cd ..

# 4. Run
streamlit run app.py
```

Open [http://localhost:8501](http://localhost:8501) and paste any message to classify it.

## Pipeline

1. **Preprocessing** — lowercase → tokenize → filter alphanumeric → remove stopwords → Porter stem
2. **Vectorization** — TF-IDF with `max_features=3000`
3. **Model** — `MultinomialNB` (scikit-learn)
4. **Serving** — Streamlit app loads cached `.pkl` artifacts
